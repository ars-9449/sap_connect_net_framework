# sap_connect_net_framework
Пример реализации коннектора к SAP для передачи данных в RFC-модуль для дальнейшей обработки данных в SAP. Бинарные данные XML передаются в поле ФМ IV_XML_X, бинарные данные файлов и названия файлов для XML передаются в поля таблицы IT_FILE.

В разработке используется проприетарный Net Connector от компании SAP.

В текущей версии используется:

- Net Connector v3.1

IDE - Visual Studio 2022.
